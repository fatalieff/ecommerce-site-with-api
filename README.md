# React API Fetch Project

This is a simple **React** project that fetches and displays data from the [Fake Store API](https://fakestoreapi.com/).  
It was created for practice purposes to learn API integration and dynamic rendering in React.

## 🚀 Features
- Fetches data using **Axios**
- Displays product images, titles, descriptions, and prices
- Includes a **search bar** to filter products
- Simple and clean UI

## 🛠️ Technologies Used
- React.js
- Axios
- CSS
